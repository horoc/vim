# VIM

## Install Plugin Manager

https://github.com/VundleVim/Vundle.vim

```shell
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

## Config 

```shell
cp vimrc ~/.vimrc
```

## Others 

command-t 

```shell
  cd ~/.vim/bundle/Command-T/ruby/command-t/
  ruby extconf.rb && make
```
